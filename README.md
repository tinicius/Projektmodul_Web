# Projektmodul Web

Welcome to the **Projektmodul Web** course repository at Hochschule München (HM).

---

## Submission Instructions

### Step 1: Fork the Repository

1. Click the **Fork** button in the top-right corner of this page
2. Select your GitHub account as the destination
3. Wait for the fork to complete

![Fork Button Location](https://docs.github.com/assets/cb-40742/mw-1440/images/help/repository/fork-button.webp)

### Step 2: Clone Your Fork

```bash
# Replace YOUR-USERNAME with your actual GitHub username
git clone https://github.com/YOUR-USERNAME/Projektmodul_Web.git
```

### Step 3: Create a Branch for Your Work (Name it as your Team number. e.g.: Team1, Team2 ... )

```bash
cd Projektmodul_Web
git checkout -b TeamX
```

### Step 4: Make Your Changes

- Add your project files to the correct team folder

### Step 5: Commit and Push Your Changes

```bash
git add .
git commit -m "Add: [Brief description of your changes]"
git push origin TeamX
```

### Step 6: Submit via Pull Request

1. Go to your forked repository on GitHub
2. Click **"Contribute"** → **"Open pull request"**
3. Ensure the base repository is `YoussefElserougi/Projektmodul_Web` and base branch is `main`
4. Fill in the pull request template with:
   - Your name / group members
   - Project title
   - Brief description
5. Click **"Create pull request"**
